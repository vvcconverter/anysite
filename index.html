VVC Player API — воспроизведение по ссылке

Плеер размещён на https://vvec-player.pages.dev/. Ниже можно подставить любую ссылку на VVC или VVC-in-MP4 и воспроизвести через этот плеер (не HTML5).

На своём сайте укажи этот же URL в iframe src.

Плеер (iframe)

Как подключить API на своём сайте

1. Вставь на страницу iframe с плеером:

<iframe id="vvcFrame" src="https://vvec-player.pages.dev/"></iframe>

2. Чтобы запустить воспроизведение по своей ссылке (с другого домена), отправь postMessage:

var frame = document.getElementById("vvcFrame");
frame.contentWindow.postMessage(
  { type: "VVC_PLAYER_COMMAND", cmd: "play", url: "https://твой-сервер.com/video.vvc" },
  "https://vvec-player.pages.dev"
);

Команды: cmd: "play" (обязательно url), cmd: "pause", cmd: "stop". Это API именно VVC Web Player (canvas + WASM), не HTML5 video.
